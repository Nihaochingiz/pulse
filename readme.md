# Fitness Tracker

Фитнес трекер для отслеживания тренировок и питания с системой очков и магазином призов.

## Возможности

- **Бег** - 10 очков за 1 км
- **Отжимания** - 1 очко за 1 раз
- **Подтягивания** - 2 очка за 1 раз
- **Гантели** - 1 очко за 10 повторений
- **Питание** - полезная еда (+5), вредная (-3)
- **Магазин призов** - покупка игр, книг, подписок за очки
- **Облачная синхронизация** - данные сохраняются в JSONBin.io
- **Экспорт в Markdown** - выгрузка отчета

## Быстрый старт

1. Склонируйте репозиторий
2. Откройте `index.html` в браузере

Или просто перейдите по ссылке GitHub Pages (после деплоя)

## Настройка облачной синхронизации (опционально)

1. Зарегистрируйтесь на [jsonbin.io](https://jsonbin.io)
2. Создайте публичный бин с данными:
```json
{
  "running": [],
  "pushups": [],
  "pullups": [],
  "weights": [],
  "meals": [],
  "points": 0,
  "purchasedItems": []
}
```
3. Скопируйте Bin ID из URL
4. Вставьте его в `script.js`:
```javascript
const BIN_ID = 'ваш_bin_id';
```

## Технологии

- HTML5
- CSS3
- JavaScript (ES6+)
- JSONBin.io API
- LocalStorage (резервное копирование)

## Структура данных

Все данные автоматически сохраняются в localStorage и синхронизируются с облаком.
